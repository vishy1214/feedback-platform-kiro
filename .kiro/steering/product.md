# Product Overview

## AI Feedback Platform

A minimal feedback collection and analysis platform that combines React frontend with FastAPI backend to process user feedback using AI.

**Core Features:**
- Feedback submission form
- Real-time feedback display
- AI-powered sentiment analysis and insights
- Topic extraction and recommendations

**Architecture:**
- Frontend: React 18 with Vite for feedback collection and display
- Backend: FastAPI for API endpoints and AI processing pipeline
- Containerized deployment with Docker Compose

**Target Use Case:**
Collecting user feedback and providing automated insights through AI analysis to help teams understand user sentiment and identify actionable improvements.
